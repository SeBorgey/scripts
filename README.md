# scripts

Набор полезных Python и Shell скриптов для автоматизации различных задач.

## Структура репозитория

### Python
- `cut_stickers.py` — нарезка стикеров.
- `del_comment.py` — удаление комментариев из кода.
- `extract_bold.py` — извлечение жирного текста.
- `html_to_md.py` — конвертер HTML в Markdown.
- `jpg_to_png.py` — конвертер изображений JPG в PNG.
- `pdf_slide_uniquer.py` — удаление дублирующихся слайдов из PDF.
- `tree.py` — вывод структуры директории в виде дерева.

### Shell
- `repair_internet.sh` — скрипт восстановления интернет-соединения.
- `repair_tablet.sh` — скрипт настройки/восстановления графического планшета.
